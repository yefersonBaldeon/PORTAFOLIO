Semana 01

FUNDAMENTOS DE LA TECNOLOGÍA WEB


¿Qué temas apredí?


Acerca de los fundamentos de las tecnologías web es un curso que nos proporciona una sólida introducción acerca de las aplicaciones y desarrollo web. Se explica que es lo que se va aprender a lo largo del curso tanto en desarrollo del front-end, como el desarrollo del back-end.

El back-end se aprenderá la lógica detrás de la escena que permite que una aplicación funcione correctamente. Aprenderemos a creer una API RESTFUL con el framework Spring Boot con Java.
Se aprenderá acerca de UI (Interfaz de Usuario) y UX (Experiencia de Usuario) son dos conceptos fundamentales para el diseño de productos digitales, como aplicaciones móviles y sitios web.

Se hablo de Visual Studio Code que es un potente editor de códigos. Es ampliamente utilizado por programadores y desarrolladores de software para escribir, depurar y administrar código en una variedad de lenguajes de programación.

Y también del generador Emmet que es un conjunto de atajos de teclado y sintaxis abreviada que se utiliza para acelerar la escritura de código.
Reflexión:
Este curso brinda una base robusta en tecnologías web, desde HTML y CSS hasta frameworks avanzados como React y Next JS. Aprender sobre UI/UX y herramientas como Visual Studio Code nos prepara para crear experiencias web atractivas y funcionales. Es un viaje hacia la maestría digital y la innovación en el desarrollo web.

Además, exploré cómo utilizar arrays para almacenar listas de elementos y objetos para estructuras de datos más complejas. Puedo escribir y probar código JavaScript directamente en la consola del navegador, lo que me permite ver resultados inmediatos. Entendí cómo el DOM permite interactuar con elementos de una página web, modificar su contenido y estructura dinámicamente, lo que es esencial para crear experiencias interactivas y dinámicas en las aplicaciones web. Estos conocimientos fundamentales en JavaScript me han proporcionado una base sólida para desarrollar aplicaciones web interactivas y dinámicas.

También me familiaricé con las funciones de flechas y funciones anónimas en JavaScript. Las funciones de flechas proporcionan una forma concisa y clara de escribir funciones, especialmente útiles para expresiones breves y legibles. Las funciones anónimas, por otro lado, son funciones sin un nombre específico y son comúnmente utilizadas como argumentos para otras funciones o asignadas a variables, proporcionando flexibilidad en el desarrollo de código.

Además, aprendí sobre la ejecución de JavaScript en el navegador y el manejo del DOM. Entendí cómo el DOM permite interactuar con elementos de una página web, modificar su contenido y estructura dinámicamente, lo que es esencial para crear experiencias interactivas y dinámicas en las aplicaciones web. Estos conocimientos fundamentales en JavaScript me han proporcionado una base sólida para desarrollar aplicaciones web interactivas y dinámicas.
Aprendí sobre el elemento canvas en HTML, una herramienta poderosa que permite crear gráficos, animaciones y visualizaciones interactivas directamente en un navegador web. Utilizando JavaScript, puedo dibujar formas, líneas y curvas en un área rectangular definida por el canvas. Este conocimiento me proporciona la capacidad de crear experiencias visuales dinámicas en sitios web, desde juegos interactivos hasta visualizaciones de datos complejas.

Semana 05

DESARROLLO APLICACIONES DE FRONTEND CON FRAMEWORKS EN JS


¿Qué temas apredí?


Aprendí acerca de las Single Page Applications (SPAs) son una revolución en el mundo del desarrollo web. A diferencia de las aplicaciones tradicionales, las SPAs no requieren que los usuarios carguen una nueva página del servidor cada vez que interactúan con la aplicación. En cambio, una SPA carga una única página HTML en el navegador y luego utiliza JavaScript para manipular y actualizar el contenido en respuesta a las acciones del usuario.

Tambien aprendí acearce de React, una poderosa biblioteca de JavaScript desarrollada por Facebook, es una opción popular para crear SPAs. La magia de React radica en su enfoque de desarrollo basado en componentes. Cada parte de la interfaz de usuario se divide en componentes reutilizables y dinámicos, lo que facilita el mantenimiento y la expansión del código.


Tambien aprendí acerca de SASS (Syntactically Awesome Stylesheets) que es un preprocesador de CSS que permite escribir estilos de una manera más organizada y eficiente. Te permite utilizar características que no están disponibles en CSS puro, como variables, anidamiento, mixins y operaciones matemáticas, lo que simplifica el proceso de escritura y mantenimiento del código CSS.
React, una biblioteca de JavaScript, revoluciona el desarrollo web al permitir la creación de interfaces interactivas en aplicaciones de una sola página. Por otro lado, Next.js, un marco de desarrollo para React, lleva esta innovación un paso más allá. Al simplificar la creación de aplicaciones web tanto del lado del servidor como del cliente, Next.js se convierte en una herramienta esencial. Ofrece características como el enrutamiento del lado del servidor y la representación en el lado del servidor, facilitando así el desarrollo de aplicaciones React robustas y eficientes que brindan una experiencia de usuario excepcional.
Reflexión:
Aprendí sobre las Single Page Applications (SPAs), revolucionando la interactividad web sin cargar páginas del servidor constantemente. Exploré React, con su enfoque modular y dinámico, y SASS para estructurar estilos eficientemente. Además, Next.js amplió mis habilidades, permitiendo aplicaciones React versátiles y excepcionales en el servidor y el cliente.
Semana 06

EVENTOS, CONDICIONALES Y BUCLES CON FRAMEWORK JS


¿Qué temas apredí?


Aprendí acerca de Vite es una herramienta de construcción de aplicaciones web extremadamente rápida y flexible que ha ganado popularidad en el ecosistema de React. A diferencia de otras herramientas de construcción como Webpack, Vite se destaca por su velocidad y eficiencia, especialmente durante el proceso de desarrollo.

Vite se integra sin problemas con React, permitiendo a los desarrolladores aprovechar sus características avanzadas mientras trabajan con la biblioteca de React. Al usar Vite con React, los desarrolladores pueden disfrutar de un entorno de desarrollo instantáneo que proporciona recargas en caliente (hot module replacement), lo que significa que los cambios en el código se reflejan inmediatamente en el navegador sin necesidad de recargar toda la aplicación. Esto mejora significativamente la eficiencia y la velocidad del flujo de trabajo de desarrollo.

Además, Vite es configurado por defecto para aprovechar las últimas características de ECMAScript (ES modules) y proporciona una estructura de proyectos clara y sencilla. Los desarrolladores pueden disfrutar de un desarrollo rápido y eficiente, lo que lo convierte en una opción atractiva para proyectos de React. En resumen, Vite es una herramienta de construcción extremadamente rápida y eficiente que se integra sin problemas con React, proporcionando un entorno de desarrollo ágil y productivo para los desarrolladores de aplicaciones web basadas en React.
Reflexión:
Vite es una herramienta veloz y flexible para construir aplicaciones web React, destacando por sus recargas instantáneas y perfecta integración. Su configuración predeterminada para ECMAScript y estructura clara lo hacen ideal, mejorando la eficiencia del desarrollo. En síntesis, Vite ofrece un entorno de desarrollo ágil y productivo para proyectos React.
